# Tradeoff Parallel Plot Example

Script: `examples/03_tradeoff_parallel_plot.py`

This example creates a synthetic multi-objective solution table, labels epsilon non-dominated solutions, applies KMeans clustering, and writes an interactive HiPlot parallel-coordinate plot.

```bash
python examples/03_tradeoff_parallel_plot.py
```

Outputs:

- `examples/_output/tradeoff_nd.csv`
- `examples/_output/tradeoff_parallel.html`

Open the HTML file in a browser to inspect the interactive plot.
