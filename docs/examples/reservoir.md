# Reservoir Sequent-Peak Example

Script: `examples/01_reservoir_sequent_peak.py`

This example generates a synthetic inflow sequence, applies `parasolpy.reservoir.sequent_peak`, and plots the resulting storage requirement.

```bash
python examples/01_reservoir_sequent_peak.py
```

Output:

- `examples/_output/sequent_peak.png`

Use this example when validating storage-sizing behavior or demonstrating how the sequent-peak result maps to required capacity.
