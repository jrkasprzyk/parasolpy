# Trace Plot Examples

Script: `examples/04_trace_plots.py`

This example generates a synthetic ensemble and creates five static plots from `parasolpy.plotting`.

```bash
python examples/04_trace_plots.py
```

Outputs:

- `examples/_output/heatmap.png`
- `examples/_output/spaghetti.png`
- `examples/_output/fan.png`
- `examples/_output/seasonality.png`
- `examples/_output/exceedance.png`

Use this example to compare heatmap, line ensemble, percentile fan, monthly seasonality, and exceedance views of the same trace ensemble.
