# Nowak Basic Examples (Notebook)

This folder is the notebook version of `example_scripts/nowak_basic_examples.py`.

## Structure
- `data/`: contextual RiverWare/Borg model files copied from `problems/MinimalExample/`
- `output/`: optional place to save figures or tables you generate from the notebook
- `nowak_basic_examples.ipynb`: the converted example notebook

## Notes
- Core simulation logic is preserved from the original script.
- The notebook demonstrates a single-year draw and a multi-trace disaggregation workflow.
