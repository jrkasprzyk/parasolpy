# File Processing Demo

This folder demonstrates how to process Borg/RiverWare solution CSV files with `borgRWhelper.file_processing`.

Contents:
- `data/NondominatedSolutions.superheader.csv`: Example with a superheader row + header row.
- `data/TradeoffAnalysis.single_header.csv`: Example with a single header row.
- `file_processing_demo.ipynb`: Notebook walkthrough.

Notebook outputs are written to `outputs/` inside this folder.
