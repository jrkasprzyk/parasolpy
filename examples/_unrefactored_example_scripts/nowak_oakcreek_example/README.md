# Nowak Oak Creek Example (Notebook)

This folder is the notebook version of `example_scripts/nowak_oakcreek_example.py`.

## Structure
- `data/oak_creek_monthly_af.xlsx`: real monthly observations and simulated annual series
- `output/`: generated outputs from the notebook
- `nowak_oakcreek_example.ipynb`: the converted example notebook

## Notes
- The notebook reads from `data/oak_creek_monthly_af.xlsx`.
- It writes simulated monthly traces to `output/oak_sim_monthly.xlsx`.
